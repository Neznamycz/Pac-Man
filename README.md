# Pac-Man
Pacman hra v pythonu jako projekt na semestriální projekt do programování. Jedná se historickou klasiku, kterou jsem přetvořil celou do pythonu! Není to tak lehké jak se zdá, tak se neboj a pojď si to vyzkoušet!

## Požadavky
- Python v 3.14.2 (Zkoušeno na této verzi) [Klikni zde pro stažení](https://www.python.org/downloads/)
- Pygame-ce (kompatibilní s python 3.14+)

## Jak hrát?
### Požadavky:
1. Stáhni si Python přes odkaz výše
2. Otevři si cmd jako administrátor
3. Napiš do cmd: pip install pygame-ce
4. Až ti řekne cmd že je vše stahnute, tak zavři a přesuň se na další krok.

### Hra:
1. Stáhni repozitář a vlož například na plochu
2. Otevři cmd
3. Zadej: cd C:\Users\default\Desktop\pacman (Vlastní cestu k dané složce kde je pacman)
4. Poté napiš: python pacman.py
5. Užívej hraní

Nebo

1. Stáhni repozitář a vlož například na plochu
2. Otevři v VS code
3. A dej "run python file"
4. Užívej hraní
